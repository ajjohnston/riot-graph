# riot-graph
A GraphQL Wrapper for the Riot API.

### Dependencies
- [Node.js v8.1.2](https://nodejs.org/en/)
- [Yarn 0.24.6](https://yarnpkg.com/en/)

### Installation
```
yarn
cp config.json.template config.json
```

### Development
```
yarn dev
```

### Testing
```
yarn test
```

### Production
```
yarn build
yarn start
```
